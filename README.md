# Маятник

- Скачать проект: 
```
git clone https://github.com/TZverev/pendulum.git
```
- Установить зависимости:
```
npm install
```
- Сделать сборку. В готовый билд хранится в папке dist. 
```
npm run-script build
```
- Открыть в браузере dist/index.html

Для запуска dev сервера:
```
npm start
```
